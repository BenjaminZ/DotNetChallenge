# .Net Challenge
This is the .Net Challenge project.

# Requirement
* [ASP.Net core 2.2](https://dotnet.microsoft.com/download).

# Usage
To run BE:

```
cd ./DotNetChallenge.Web
dotnet restore
dotnet run
```

To run FE:

```
cd ./DotNetChallenge.Web
npm ci
npm run start
```

# About solution
One test project and one web project since it's not a complex requirement.
